    class Assessment

    This class represents one assessment with the attribute points. Also this class contains many exams and belongs to a user, rank and a topic instance. It derived from the base class ActiveRecord::Base. Therefore an instance of this class provides all CRUD (create, read, update and delete) operations.

    This method will return the whole number of possible points for this assessment.

def available_points?
  availablePoints = 0
  self.exams.each do |exam|
    availablePoints += exam.wbt.getDifficultyPoints
  end
  availablePoints
end
